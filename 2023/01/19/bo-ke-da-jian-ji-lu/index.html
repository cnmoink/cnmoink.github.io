部署

可选GitHub Pages和Cloudflare Pages，如有自己域名，推荐Cloudflare Pages
这两个部署方式均需要GitHub

流程

创建私有库

创建链接
仓库名任选
勾选Private

GitHub Pages部署方式

生成GitHub Token

打开 https://github.com/settings/tokens

私有库设置

打开创建私有库的设置
https://github.com/GitHub用户名/你创建的私有库名/settings/secrets/actions
添加密钥HEXO_DEPLOY，值为你的刚刚生成的Token

创建公开库

库名:修改为自己GitHub用户名.github.io

部署自动流程

打开本地博客目录，新建.github/workflows目录，创建并编辑.github/workflows/hexo-deploy.yml

内容：

完成部署

打开本地博客目录

等待2分钟左右，访问用户名.github.io即可

Cloudflare Pages部署方式

注册账户，进入控制台，打开Pages页面
连接到GitHub，登录账户，在GitHub 中选择生效的库为你创建的私有库
部署命令填为：hexo generate
环境变量添加：
NODE_VERSION 值 16.18.1
PYTHON_VERSION 值 3.7

打开本地博客目录

等待2分钟左右打开Cloudflare给出站点查看效果
